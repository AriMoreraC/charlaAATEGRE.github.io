Crear un Contacto

Aprende a guardar números y hacer llamadas fácilmente

Guardar un Número

  1. 1. Abra la aplicación Contactos o Teléfono.
  2. 2. Toque el botón + o "Agregar contacto".
  3. 3. Escriba el nombre de la persona.
  4. 4. Escriba el número de teléfono.
  5. 5. Presione Guardar.
  6. Revise que el número esté correcto antes de guardar.

Llamar a un Contacto
  1. 1. Abra la aplicación Teléfono o Contactos.
  2. 2. Busque el nombre del contacto.
  3. 3. Toque el contacto.
  4. 4. Presione el botón de llamar .
  5. 5. Espere a que la persona conteste.

Editar o Eliminar

  1. 1. Abra el contacto guardado.
  2. 2. Toque Editar para cambiar datos.
  3. 3. Para eliminar, busque la opción "Eliminar".
  4. Tenga cuidado al eliminar, no se puede recuperar fácilmente.
